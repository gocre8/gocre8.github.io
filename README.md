# gocre8.github.io

Interactive web tools by Geoffrey Owen Miller, hosted on GitHub Pages and embedded on [geoffreyowenmiller.com](https://www.geoffreyowenmiller.com).

---

## Active Tools

### Color Tools

| File | Tool | Description | Embedded On |
|------|------|-------------|-------------|
| `index.html` | Mood Palette | Extracts 5 dominant colors from an uploaded image using k-means clustering. Returns Hex, RGB, HCT, CMYK, and LCH values. Bilingual (EN/ES). | [/colorcoding](https://www.geoffreyowenmiller.com/colorcoding) |
| `color-mixer-lab.html` | LAB Color Mixer | Mixes two colors in LAB or Kubelka-Munk (RGB-based) space. Shows mixing path on a 2D LAB graph. | [/colorcoding](https://www.geoffreyowenmiller.com/colorcoding) |
| `colorspace-viewer.html` | Color Space Explorer | 3D interactive viewer of RGB, HSL, and HSV color models using Three.js. Orbit, slice, and pick colors. | [/colorspaceexplorer](https://www.geoffreyowenmiller.com/colorspaceexplorer), [/colorcoding](https://www.geoffreyowenmiller.com/colorcoding) |
| `pixel-grid-quantizer.html` | Pixel Grid Quantizer | Quantizes an image into a color grid. | [/chromagrid](https://www.geoffreyowenmiller.com/chromagrid), [/colorcoding](https://www.geoffreyowenmiller.com/colorcoding) |
| `munsell-explorer.html` | Munsell Explorer | 3D visualization of the Munsell color system using Three.js. | — |

### Poetry Tool

| File | Tool | Description | Embedded On |
|------|------|-------------|-------------|
| `poem_grid_tool_v3_23.html` | Phonetic Grid Tool | Upload a poem photograph → OCR extracts letters → color grid based on phonetic categories (stops, fricatives, sonorants). Uses Cloudflare proxy for Claude OCR. | [/poemcolor](https://www.geoffreyowenmiller.com/poemcolor) |

### Nature Guides

| File | Tool | Description | Embedded On |
|------|------|-------------|-------------|
| `aquarium_leaf_guide2.html` | Aquarium Leaf Guide | Reference guide for aquarium botanicals — 16 leaf types with SVG illustrations, NYC foraging guide, sourcing table. | [/aquariumleaves](https://www.geoffreyowenmiller.com/aquariumleaves) |
| `pond_microbe_guide.2.html` | Pond Microbe Guide | Field guide to pond microorganisms — protozoa, algae, micro-animals, bacteria, and invertebrates with SVG illustrations and microscopy tips. | [/pondlife](https://www.geoffreyowenmiller.com/pondlife) |
